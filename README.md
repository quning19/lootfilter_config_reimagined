### 功能清单

1. 杂项隐藏
   1. 低等药剂 { "hp1", "hp2", "hp3", "hp4", "hp5", "mp1", "mp2", "mp3", "mp4", "mp5" }
   2. 卷轴 { "tsc", "isc"}
   3. 箭 {"aqv","cqv"}
2. 重要信息显示
   1. 装备等级[X], [E]显示
   2. 超强显示[S]
   3. 符文之语提示
   4. 显示当前孔数量
   5. 描述显示最大孔数量 itype={45,50,10,12}
3. 重要物品显示
   1. 高级符文显示
   2. Orb显示
   3. boss钥匙 & 精华
   4. 任务物品
4. 白色品质装备显示
   1. 破碎装备隐藏
   2. 隐藏1孔物品
   3. 通用头、胸、盾
      1. 精英 超强：高亮 200
      2. 特定 精英：高亮 255 & 通知
      3. 扩展 超强：高亮 150
      4. 普通 + 扩展 非超强：隐藏
   4. 手套、鞋、腰带：
      1. 普通 + 扩展：隐藏
   5. 高亮职业装备
      1. 圣骑士盾 4r40
      2. 超强弓 4孔以上
      3. 2/3孔 Orb
      4. 3孔刺客爪
   6. 通用武器显示
      1. 0,3,4,5,6孔水晶剑
      2. 0,4,5孔连枷
      3. 0,5,6孔幻化
      4. 5孔狂战士斧
      5. 0,4孔镰刀
      6. 0,4,5,6孔法杖/长矛
   7. 武器隐藏
      1. 普通+扩展 隐藏：除orb, staff, 镰刀, 连枷，水晶剑，刺客爪
      2. 上限5,6孔精华武器，2,3孔: 隐藏
5. 无形装备显示
   1. 高亮3/4孔精华胸甲，头甲，盾牌
   2. 高亮3/4/5/6孔单手精华武器：单手剑、狂战士斧、
   3. 高亮4/5/6孔双手精华武器：长柄、矛
6. 蓝色品质装备显示
   1. 隐藏蓝色品质普通、稀有装备
   2. 显示蓝色charm,jewel,ring, amulet,
7. 金色品质装备显示
   1. 高亮显示手套、鞋、
   2. 高亮显示Ring、Amulet、Charm、jewel
   3. 隐藏除手套、鞋、Orb、Wand非精华装备
8. 暗金装备显示
   1. 所有暗金装备
   2. 暗金charm
   3. 暗金Jewel
9. Set套装显示
   1. 所有套装装备
